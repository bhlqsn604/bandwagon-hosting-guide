## 一、先搞清楚：个人建站到底需要什么

在挑套餐之前，得先想明白你建的是个什么站。这决定了你该把钱花在线路上还是配置上。

- **个人博客/技术笔记站**：流量不大，日均几十到几百IP，1GB内存跑WordPress绰绰有余，关键是访问速度要稳，别让读者等。
- **作品集/静态站**：更轻量，256MB内存都能跑，重点是不宕机、能访问。

## 八、常见问题 FAQ

**Q：搬瓦工建站需要备案吗？**
不需要。服务器在境外，不用国内备案，域名解析过去就能访问。

**Q：1GB内存跑WordPress够吗？**
够。WordPress本身不重，加个缓存插件（比如WP Super Cache），1GB内存跑日均几百IP的博客很轻松。装了宝塔面板的话，面板本身要吃点内存，建议至少1GB起步。

**Q：CN2 GIA和CN2 GIA-E有什么区别？**
CN2 GIA是电信的优质回国线路本身；GIA-E（GIA-Economy/Enhanced）是搬瓦工基于CN2 GIA做的多机房套餐方案，可选机房更多、性价比更好，是个人建站最常用的版本。

**Q：限量版套餐值得买吗？**
价格是真香，但要注意限制：有的不能迁移机房（比如Fremont款），有的库存不稳定、随时可能下架。适合当主力站的备份方案或纯练手用，不建议拿来做重要的商业站。

**Q：晚高峰卡怎么办？**
先确认是不是线路问题——在KiwiVM里看流量统计和节点状态。如果当前机房线路拥堵，CN2 GIA-E及以上套餐支持一键迁移机房，换个节点往往就顺了。
